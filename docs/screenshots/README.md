## Taking Screenshots

1. Run the application:
   ```bash
   # Backend
   python -m uvicorn app.main:app --reload

   # Frontend
   cd frontend/study-planner-pro
   npm run dev
   ```

2. Navigate to http://localhost:5173

3. Take screenshots of each page using:
   - Windows: Windows Key + Shift + S
   - Mac: Cmd + Shift + 4
   - Linux: Screenshot tool or PrintScreen

4. Save screenshots with the exact names listed above

5. Optimize images for web:
   - Recommended size: 1280x800 or 1920x1080
   - Format: PNG for UI screenshots
   - Compress using tools like TinyPNG
